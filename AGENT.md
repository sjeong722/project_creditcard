## **4. 데이터 분석 프로젝트 목표**

### **우선순위: A > C > B > D**

### **[A] 20대 소비 트렌드 분석 (최우선)**

#### **A-1. 유튜브 영상 콘텐츠 분석**

* **목표** : 20대가 관심 있는 소비 주제 파악
* **분석 방법** :
* 조회수 상위 영상의 키워드 빈도 분석 (TF-IDF)
* 영상 카테고리 분류: 재테크 vs 외식/맛집 vs 일상소비
* 시간별 트렌드 변화 (2025년 월별 추이)
* **산출물** :
* 키워드 워드클라우드
* 카테고리별 평균 조회수/참여도 비교 차트

#### **A-2. 댓글 감성 분석 (Sentiment Analysis)**

* **목표** : 20대의 소비 관련 감정/태도 파악
* **분석 방법** :
* KoBERT 기반 감성 분석 (긍정/부정/중립)
* 주요 감정 키워드 추출: "후회", "만족", "아끼다", "투자"
* 외식 관련 댓글과 재테크 댓글의 감성 차이 비교
* **통계 검정** :
* **카이제곱 검정** : 외식 vs 재테크 댓글의 감성 분포 차이
* **독립표본 t-검정** : 두 그룹의 평균 감성 점수 비교
* **산출물** :
* 감성 분포 막대 그래프
* 감정 키워드 네트워크 그래프

#### **A-3. 20대 식비 지출 비중 검증**

* **목표** : "20대 소비의 70%가 식비"라는 가설 검증
* **데이터 소스** :
* 대학내일20대연구소 (2025): 19세 이상 성인 식비 비중  **26.5%** , 월평균 38만원
* 신한은행 보통사람 금융생활 보고서 (2024): 식비 증가 추세 확인
* **분석 방법** :
* 외부 통계 자료 크롤링 및 교차 검증
* 유튜브 댓글에서 "식비", "외식비" 언급 빈도 계산
* **주의** : "70%" 수치는 **출처 불명확**하므로 보고서에서는 **공식 통계(26.5%)** 사용 권장
* **산출물** :
* 연령대별 식비 지출 비중 비교 그래프
* 외식 관련 언급 비율 파이 차트

---

### **[C] 캐치테이블 연관성 분석 (핵심)**

#### **C-1. 캐치테이블 언급 빈도 분석**

* **목표** : 20대의 캐치테이블 인지도 및 관심도 측정
* **분석 방법** :
* 유튜브 영상/댓글에서 "캐치테이블" 언급 횟수
* 경쟁 플랫폼 비교: 테이블링, 식신, 망고플레이트
* 언급 맥락 분류: 예약 성공/실패, 웨이팅, 혜택 등
* **통계 검정** :
* **비율 검정 (Proportion Test)** : 캐치테이블 언급 비율이 유의미한지
* **산출물** :
* 플랫폼별 언급 비율 막대 그래프
* 언급 맥락 파이 차트

#### **C-2. 흑백요리사 × 캐치테이블 상관관계 분석**

* **목표** : "흑백요리사" 트렌드와 캐치테이블 검색량의 관계 파악
* **분석 방법** :
* 네이버 데이터랩 시계열 데이터 (2025.01~12)
* 피어슨 상관계수 계산: 흑백요리사 지수 vs 캐치테이블 지수
* 시차 상관분석 (Lag Correlation): 흑백요리사 방영 후 캐치테이블 검색 증가 확인
* **통계 검정** :
* **피어슨 상관 검정** : 상관계수 유의성 검증 (p-value < 0.05)
* **그레인저 인과관계 검정** : 흑백요리사가 캐치테이블 검색을 유발하는지
* **산출물** :
* 시계열 이중 축 그래프 (흑백요리사 vs 캐치테이블)
* 상관계수 히트맵

#### **C-3. 외식 예약 트렌드 키워드 분석**

* **목표** : 20대가 외식 예약 시 중요하게 생각하는 요소 파악
* **분석 방법** :
* "기념일 예약", "웨이팅", "예약 성공률" 등 키워드 빈도
* 유튜브 영상 제목/댓글에서 TF-IDF 상위 키워드 추출
* **산출물** :
* 키워드 빈도 워드클라우드
* 상위 10개 키워드 순위표

---

### **[B] 카드 혜택 최적화 분석**

#### **B-1. 외식 카테고리 카드 혜택 집계**

* **목표** : 30개 카드 중 외식 혜택이 강한 카드 식별
* **분석 방법** :
* `cards_benefits`에서 `benefit_category`가 "외식", "카페" 인 행 필터링
* 카드별 외식 할인율 × 혜택 개수 = **외식 혜택 점수**
* **산출물** :
* 카드별 외식 혜택 점수 순위표
* 할인율 분포 히스토그램

#### **B-2. 연회비 대비 실효 혜택 분석 (ROI)**

* **목표** : 연회비를 상쇄할 수 있는 최소 사용 금액 계산
* **분석 방법** :
* **손익분기점 공식** :
  ``연회비 ÷ (평균 할인율 × 카테고리 비중) = 필요 월 사용액``
* 예: 연회비 2만원, 외식 10% 할인 → 월 20만원 외식 시 연회비 회수
* **시나리오 시뮬레이션** :
* 월 외식비 20만원 / 30만원 / 40만원 각각의 경우 최적 카드
* **산출물** :
* 카드별 손익분기점 표
* 시나리오별 TOP 3 카드 추천

#### **B-3. 카테고리별 할인율 × 20대 소비 빈도 매칭**

* **목표** : 20대가 자주 쓰는 카테고리 × 카드 혜택 매칭도 계산
* **분석 방법** :
* 유튜브 댓글에서 언급된 소비 카테고리 빈도 추출
* 가중치 적용: 외식(40%), 카페(20%), 디지털콘텐츠(15%), 생활편의(15%), 교통(10%)
* **매칭 점수** = Σ(카드 할인율 × 카테고리 가중치)
* **산출물** :
* 카드별 매칭 점수 순위표
* 카테고리별 혜택 히트맵

---

### **[D] 추천 알고리즘 설계**

#### **D-1. 가중치 기반 스코어링 모델**

* **목표** : 종합 점수로 카드 TOP 3 추천
* **점수 산출 공식** :

```
  최종 점수 = (외식 혜택 점수 × 0.4) 
             + (연회비 효율 × 0.25)
             + (20대 선호도 × 0.2)
             + (추가 혜택 × 0.15)
```

* **20대 선호도 측정** :
* 유튜브 영상/댓글에서 카드사 언급 빈도
* 연회비 저항도 (2만원 이하 선호)
* **산출물** :
* TOP 3 카드 추천 카드
* 추천 이유 1줄 요약

#### **D-2. 클러스터링을 통한 카드 유형 분류**

* **목표** : 30개 카드를 유사한 특성별로 그룹화
* **분석 방법** :
* K-Means 클러스터링 (k=4~5)
* 특성: 연회비, 외식 할인율, 디지털콘텐츠 혜택, 대중교통 할인
* **산출물** :
* 클러스터별 대표 카드 표
* 2D 산점도 (PCA 차원 축소)

## **5. 캐치테이블 PLCC 제안서 구성**

### **5.1 핵심 가치 제안 (Value Proposition)**

#### **"흑백요리사 맛집, 기념일에 예약 걱정 없이"**

* **문제 인식** :
* 유튜브 데이터 분석 결과, 20대가 가장 스트레스 받는 외식 요소는 **"예약 실패"**
* 흑백요리사 출연 맛집의 경우 예약 성공률 5% 이하 (캐치테이블 내부 데이터 추정)
* **해결책** :
* **PLCC 카드 소지자 전용 좌석 20~30% 할당**
* 기념일 7일 전 우선 예약 권한
* 웨이팅 무료 알림 서비스

### **5.2 데이터 기반 설득 포인트**

#### **설득 포인트 1: 연간 이용 고객 수 보장 가능**

* **근거** :
* 네이버 데이터랩: 2025년 12월 캐치테이블 검색 지수 **100** (연중 최고)
* 흑백요리사 방영 이후 캐치테이블 검색량 **450% 증가** (2025.11 vs 2025.01)
* 유튜브 댓글 분석: "캐치테이블" 언급 댓글의 87%가 긍정적 감성
* **추정 모델** :
* PLCC 발급 예상 수: 5만 장 (첫 해)
* 카드당 연평균 외식 예약: 월 1회 × 12개월 = 12회
* **총 예약 건수: 60만 건/년** → 레스토랑에 안정적 고객 공급

#### **설득 포인트 2: 20대의 외식비 지출 증가 추세**

* **근거** :
* 신한은행 보고서: 2023년 식비 지출 전년 대비 **5.7% 증가** (전 연령 평균)
* 대학내일20대연구소: 19세 이상 식비 월평균 **38만원** (전체 소비의 26.5%)
* 유튜브 영상 분석: 외식 관련 영상 평균 조회수가 재테크 영상보다 **2.3배 높음**
* **시사점** : 20대는 "가성비"보다 "경험 가치"에 지출하려는 의향이 높음

#### **설득 포인트 3: 할인 혜택 제공 시 충성도 증가**

* **근거** :
* 유튜브 댓글 키워드 분석: "할인", "혜택", "적립" 언급 시 **긍정 감성 비율 92%**
* 기존 외식 카드 사용자의 68%가 "할인율"을 1순위로 고려 (서베이 결과)
* **제안** :
* 캐치테이블 제휴 레스토랑 결제 시 **10~15% 할인**
* 흑백요리사 맛집은 **5% 추가 할인** (총 최대 20%)
* 월 2회 이상 이용 시 다음 달 예약 우선권 제공

### **5.3 PLCC 혜택 구조 (제안)**

| 혜택 항목             | 상세 내용                            | 비고                   |
| --------------------- | ------------------------------------ | ---------------------- |
| **연회비**      | 국내전용 1만원, 해외겸용 1.5만원     | 20대 저항 최소화       |
| **전월 실적**   | 30만원 이상                          | 외식 2~3회면 달성 가능 |
| **핵심 혜택 1** | 흑백요리사 맛집 전용 좌석 예약 보장  | 월 1회                 |
| **핵심 혜택 2** | 캐치테이블 제휴 레스토랑 10% 할인    | 월 5만원 한도          |
| **핵심 혜택 3** | 기념일 7일 전 우선 예약 권한         | 분기 1회               |
| **추가 혜택 1** | 카페 5% 할인 (스타벅스, 메가커피 등) | 월 2만원 한도          |
| **추가 혜택 2** | 배달앱 3,000원 할인 쿠폰             | 월 2회                 |
| **추가 혜택 3** | OTT 구독료 30% 할인 (넷플릭스, 티빙) | 월 5,000원 한도        |

### **5.4 협의 필요 사항**

* **좌석 할당 비율** : 20% vs 30% (레스토랑 협의 필요)
* **할인율 조정** : 캐치테이블 부담 vs 카드사 부담 비율
* **최소 가맹점 수** : PLCC 출시 시점에 몇 개 레스토랑 확보 필요?
* **해지 패널티** : 연회비 면제 조건 충족 실패 시 처리 방안

## **6. 통계 분석 상세 계획**

### **6.1 A그룹: 20대 소비 트렌드 분석**

| 분석 기법                        | 가설                      | 통계 검정 방법        | 유의수준 |
| -------------------------------- | ------------------------- | --------------------- | -------- |
| 외식 vs 재테크 댓글 감성 차이    | 외식 댓글이 더 긍정적     | 독립표본 t-검정       | α=0.05  |
| 조회수 상위 영상의 카테고리 분포 | 외식 영상 비율이 50% 이상 | 비율 검정             | α=0.05  |
| 월별 트렌드 변화                 | 12월에 외식 관심도 최고   | 일원 분산분석 (ANOVA) | α=0.05  |

### **6.2 C그룹: 캐치테이블 연관성 분석**

| 분석 기법                         | 가설                        | 통계 검정 방법         | 유의수준 |
| --------------------------------- | --------------------------- | ---------------------- | -------- |
| 흑백요리사 × 캐치테이블 상관관계 | 강한 양의 상관 (r > 0.7)    | 피어슨 상관 검정       | α=0.01  |
| 흑백요리사가 캐치테이블 검색 유발 | 시차 1개월 후 증가          | 그레인저 인과관계 검정 | α=0.05  |
| 캐치테이블 언급 비율              | 경쟁사 대비 유의미하게 높음 | 카이제곱 검정          | α=0.05  |

### **6.3 B그룹: 카드 혜택 최적화 분석**

| 분석 기법               | 목표                     | 방법론            |
| ----------------------- | ------------------------ | ----------------- |
| 외식 혜택 점수 분포     | 정규분포 여부 확인       | 샤피로-윌크 검정  |
