# 데이터 사전(초안)

## cards_info (카드 정보)

- `card_id`: 카드 고유 ID
- `card_name`: 카드명
- `issuer`: 카드사명
- `annual_fee_domestic`: 국내전용 연회비(원)
- `annual_fee_overseas`: 해외겸용 연회비(원)
- `min_usage_requirement`: 최소 이용 실적
- `card_image_url`: 카드 이미지 URL
- `detail_page_url`: 상세 페이지 URL

## cards_benefits (카드 혜택)

- `card_id`: 카드 ID
- `benefit_order`: 혜택 순서
- `benefit_category`: 혜택 카테고리(외식/카페/생활편의 등)
- `discount_type`: 할인/적립 구분
- `discount_rate`: 할인율(%)
- `is_optional`: 선택 혜택 여부
- `benefit_summary`: 혜택 요약
- `benefit_detail`: 혜택 상세
